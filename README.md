# 🛒 DigiTools Web Application

## 📌 Project Name

DigtTools Web Application

Live Demo🔗: https://digi-tools-zone.vercel.app/

---

## 📝 Description

A responsive and interactive product cart system where users can browse products, add them to cart, remove items, and proceed to checkout. The UI is designed based on Figma and focuses on clean design and smooth user experience.

---

## ⚙️ Technologies Used

* React.js
* Tailwind CSS & DaisyUI
* JavaScript (ES6+)
* React-Toastify
* JSON

---

## ✨ Key Features

1. 🛒 Add to cart & remove functionality with live update
   Users can easily add products to the cart by clicking the "Buy Now" button. The cart updates instantly without reloading the page. Each item in the cart can be removed individually using a remove button, and the total product count in the navbar updates in real-time.

2. 🔄 Product & Cart toggle system
   The application includes two toggle buttons that allow users to switch between the Product section and the Cart section. By default, the Product section is visible. When the Cart button is clicked, it displays all selected items or an empty message if no products are added.

3. 🔔 Toast notifications for user actions (add, remove, checkout)
   The app uses React-Toastify to provide instant feedback to users. When a product is added to the cart, removed, or when the user proceeds to checkout, a toast notification appears to confirm the action, improving user experience and interaction.
